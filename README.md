# climate-challenge-week0

# Climate Analysis (2015–2026)

This project analyzes temperature and rainfall patterns across five African countries: Ethiopia, Kenya, Sudan, Tanzania, and Nigeria using NASA POWER climate data.

## 📁 Project Structure

* notebooks/

  * ethiopia_eda.ipynb
  * all_countries_eda.ipynb

## Key Insights

* Clear seasonal temperature patterns across all countries
* Rainfall distribution is highly skewed (many dry days, few heavy rains)
* Sudan shows the highest exposure to extreme heat
* Climate vulnerability varies significantly across countries

## Methodology

* Data cleaning (handling missing values)
* Feature engineering (DATE, MONTH, Country)
* Exploratory Data Analysis (EDA)
* Cross-country comparison
* Climate vulnerability scoring

## Data Source

NASA POWER (MERRA-2)
